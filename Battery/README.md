# Battery on HackberryPi_CM5

The battery used on HackberryPi_CM5 is Lipo battery with multiple certifications

```Dimension```: 83x50.5x10.1mm  
```Capacity```: 5000mah @ 3.7V  
```Battery Voltage```: 2.75-4.2V  
```Weight```: 84.4g
